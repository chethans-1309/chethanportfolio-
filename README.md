Portfolio — Chetan S

Futuristic, glass/neon themed portfolio built with Next.js 14, Tailwind CSS, and Framer Motion.

Local Development
```
npm install
npm run dev
```

Add resume: put file at public/sodaresume.pdf

Add profile photo
- Save your image as `public/images/profile.jpg` (recommended ~600x600).
- The hero automatically shows it; use a different filename by updating `components/Hero.tsx`.

Tuning
- RainOverlay config props control intensity.
- Update transition durations/easing in app/template.tsx and component motion props.

Deploy
- Use Vercel. Build: next build.
